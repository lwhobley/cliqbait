### GLITCH MOTION AI BACKGROUND SYSTEM

Create a reusable <GlitchBackground /> React component that renders behind all page content using `position: fixed; z-index: -1; inset: 0`. It accepts a `variant` prop:
- "intro" — most intense, used on landing intro sequence
- "hero" — subtle, used on storefront hero sections
- "product" — minimal grain only
- "cart" — off (white background only)

**EFFECT 1 — Chromatic Aberration RGB Split (WebGL Shader via OGL)**
On intro and hero sections, render a fullscreen WebGL canvas using OGL with a GLSL fragment shader that:
- Splits R, G, B channels horizontally by different offsets animated via `uTime` uniform
- R channel: +8px offset, B channel: −8px offset, G channel: no offset
- Vertical scan line jitter: every 80–120ms randomly shift a horizontal band ±5px
- Glitch fires in bursts: 200ms intense → 2s calm → repeat (randomized timing)

Fragment shader:
```glsl
uniform sampler2D uTexture;
uniform float uTime;
uniform float uGlitchIntensity;
varying vec2 vUv;
void main() {
  float offset = sin(uTime) * uGlitchIntensity * 0.01;
  float r = texture2D(uTexture, vUv + vec2(offset, 0.0)).r;
  float g = texture2D(uTexture, vUv).g;
  float b = texture2D(uTexture, vUv - vec2(offset, 0.0)).b;
  gl_FragColor = vec4(r, g, b, 1.0);
}
```

**EFFECT 2 — Cursor-Reactive Fluid Gradient (Three.js + GLSL)**
On hero and home sections, render a fullscreen Three.js PlaneGeometry with custom ShaderMaterial:
- Tracks mouse via `mousemove` → `uMouse` vec2 uniform (normalized 0–1)
- fbm (fractal Brownian motion) noise generates fluid distortion field in GLSL
- Gradient stays monochromatic: pure black (#000000) to deep charcoal (#111111) — NEVER color
- Cursor creates lens-distortion ripple: radius ~200px, strength 0.03
- Mobile fallback: auto-panning slow wave using `uTime` only
- Mouse lerp factor: 0.08 for smooth lag

```javascript
const uniforms = {
  uTime: { value: 0 },
  uMouse: { value: new THREE.Vector2(0.5, 0.5) },
  uResolution: { value: new THREE.Vector2(window.innerWidth, window.innerHeight) }
};
// useFrame: uniforms.uTime.value += 0.005
// mousemove: lerp uMouse toward normalized cursor position
```

**EFFECT 3 — Film Grain Noise Overlay**
Fullscreen `<canvas>` fixed overlay on ALL pages: `z-index: 9999, pointer-events: none, opacity: 0.04`:
- requestAnimationFrame at 24fps filling canvas with randomized ImageData pixel noise
- Product pages: reduce to opacity 0.025 so product images stay clean
- pointer-events: none required throughout

```javascript
function renderGrain(canvas) {
  const ctx = canvas.getContext('2d');
  const imageData = ctx.createImageData(canvas.width, canvas.height);
  for (let i = 0; i < imageData.data.length; i += 4) {
    const v = Math.random() * 255;
    imageData.data[i] = imageData.data[i+1] = imageData.data[i+2] = v;
    imageData.data[i+3] = 255;
  }
  ctx.putImageData(imageData, 0, 0);
}
```

**EFFECT 4 — CSS Glitch Text Bursts**
On brand name stamp and section headers scroll-into-view:
- `::before` and `::after` pseudo-elements with `clip-path: inset()` create offset ghost copies in red (#FF0000) and cyan (#00FFFF)
- `.is-glitching` CSS class toggled by Framer Motion `onAnimationStart`
- Maximum glitch duration: 800ms — always resolves to clean readable black

```css
@keyframes glitchClip {
  0%   { clip-path: inset(20% 0 50% 0); transform: translate(-4px, 0); }
  25%  { clip-path: inset(60% 0 10% 0); transform: translate(4px, 2px); }
  50%  { clip-path: inset(10% 0 70% 0); transform: translate(-2px, -2px); }
  75%  { clip-path: inset(40% 0 30% 0); transform: translate(3px, 1px); }
  100% { clip-path: inset(0% 0 0% 0);   transform: translate(0, 0); }
}
```

**EFFECT 5 — Scan Line Sweep (Intro Only)**
During product flash reel: a 2px white line `position: absolute` animates `top: -2px → 100vh` over 1.2s linear, looping infinitely at opacity 0.15. Pure CSS animation, no JS.

**Performance Rules:**
- All WebGL canvases: `alpha: false`, `antialias: false`
- Pause all rAF loops on `visibilitychange` hidden
- `prefers-reduced-motion`: disable all glitch/grain effects, fallback to static black
- Throttle mousemove to 60fps via `lastTime` timestamp check
- Mobile (< 768px): disable WebGL shaders entirely, keep only CSS grain at 0.02 opacity

- Cancel: redirect back to cart

### PRINTFUL API SERVICE (`printfulService.ts`)
- `GET /api/products` — all synced store products
- `GET /api/products/:id` — single product + all variants + real-time pricing
- `GET /api/products/category/:slug` — filter by category
- `POST /api/orders` — create Printful fulfillment order (called by Stripe webhook) with: recipient (name, address from Stripe session), items (sync_variant_id + quantity)
- Cache product listings in PostgreSQL for 60 minutes to reduce API calls
- API key stored in Replit Secrets as PRINTFUL_API_KEY
